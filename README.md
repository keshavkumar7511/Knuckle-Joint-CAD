# Knuckle Joint CAD Project

## Software Used
SolidWorks

## Project Description
Designed and assembled a knuckle joint used for transmitting tensile loads between two rods while allowing angular movement.

## Skills Demonstrated
- 3D CAD Modeling
- Assembly Design
- Engineering Drawings
- Mechanical Component Design

- SolidWorks Part Files (.SLDPRT)
